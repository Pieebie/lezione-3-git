📌 Come aggiungere il tuo nome

  1. Clona il repo oppure fai il fork se non hai accesso diretto.
  2. Crea un branch: git switch -c feat/aggiungi-nome-tuonome
  3. Apri index.html e cerca il commento AGGIUNGI QUI IL TUO NOME.
  4. Copia un blocco <li> esistente, incollalo subito sopra il commento e modifica i dati.
  5. Fai git add index.html, poi git commit -m "feat: aggiungi Nome Cognome".
  6. Esegui git push e apri una Pull Request su GitHub!
